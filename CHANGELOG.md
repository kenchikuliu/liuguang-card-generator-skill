# 流光卡片生成器 - 更新日志

## v1.4.1 (2026-03-22)

### 🇬🇧 charlii Card System 英文增强版

**新增文件：**
- `CHARLII_SPEC_EN_v1.0.md` - 英文卡片专用规范（非中文直译）

**核心区别：**
- 英文卡片 = 判断切句，不是信息压缩
- 防「读起来像翻译稿」：rewrite not translate
- 字数标准：60–110词（安全区），130+词（风险区）
- 单行上限：10词（比中文更严格）
- 新增 punchline rhythm 要求
- 新增 filler word 删除清单
- 新增 Card Writing Formula（Hook/Shift/Judgment/Framework/Punchline）
- 新增 Quick Prompt 模板（可直接给模型用）

## v1.3.0 (2026-03-21)

### 🌍 核心新功能：双语卡片生成

**新增功能：**
- ✅ **双语支持**：输入 1 篇博客 → 输出中文版 + 英文版各 6 张卡片
- ✅ **自动翻译**：使用 Claude API 翻译内容，保持 Markdown 格式
- ✅ **语言自动检测**：自动识别输入是中文还是英文
- ✅ **社交文案生成**：为每套卡片生成平台优化的文案
  - 中文版：小红书（emoji 丰富）+ 微博（热点话题）
  - 英文版：Twitter（简洁专业）+ LinkedIn（专业深度）
- ✅ **分语言输出目录**：`/tmp/cards/zh/` 和 `/tmp/cards/en/`
- ✅ **英文卡片容量调整**：英文版每张约 400 字节（中文 310 字节）

**新增文件：**
- `translator.py` - 语言检测和翻译模块
- `social_copy_generator.py` - 社交媒体文案生成模块

**修改文件：**
- `generator.py` - 重构支持双语，新增 `_generate_single_language()`
- `image_handler.py` - 添加 `language` 参数支持

**输出格式：**
```
/tmp/cards/
├── zh/
│   ├── card_01.png ~ card_06.png
│   └── social_copy.json  # 小红书 + 微博文案
└── en/
    ├── card_01.png ~ card_06.png
    └── social_copy.json  # Twitter + LinkedIn 文案
```

## v1.2.0 (2026-03-20)

### 🎯 核心改进：博客文章转卡片

**新增功能**：
- ✅ **图片自动处理**：自动检测并移除文章中的图片（Markdown/HTML 格式）
- ✅ **智能内容分割**：改进分割算法，每张内容卡片约 310 字节，分布更均匀
- ✅ **纯文字卡片**：内容卡片只显示文字，不嵌入原文图片
- ✅ **动态平衡算法**：确保 5 张内容卡片的字节数均匀分配

**技术细节**：
- 新增 `image_handler.py` 模块处理图片检测和移除
- 优化 `_split_text_only()` 方法，使用动态平衡策略
- `base_capacity` 从 325 字节调整到 310 字节
- 支持更长的博客文章（1000-1500 字）

**测试结果**（基于 1442 字节测试文章）：
```
卡片 1（封面）: AI 生成图片 + 标题
卡片 2: 277 字节
卡片 3: 273 字节
卡片 4: 276 字节
卡片 5: 268 字节
卡片 6: 348 字节
平均: 288 字节/卡片
```

**使用场景**：
- 博客文章转小红书/微信图文
- 长文内容可视化
- 知识笔记卡片化
